# LapClip Visualizer — iPhone / iPad（Safari ブックマークレット）

iOS では拡張機能の配布が難しいため、**Safari の「お気に入り（ブックマーク）」として保存して1タップで起動する**方式で提供します。PC のブックマークレットと**同じコード**で、公式 LapClip ページを Safari で開いた状態で実行すると、そのページ上に F1 風オーバーレイを重ねます。**外部送信・保存・再配信はしません**（ページの DOM を読むだけ）。

> ⚠️ 非公式。LapClip / マトリックス社 / Tour of Japan / 大会運営とは無関係。公式結果は必ず LapClip で確認してください。

## なぜこの手順なのか
iOS Safari は **アドレスバーに `javascript:` を直接入力しても実行できません**。必ず「ブックマークとして保存 → URL 欄にコードを貼り付け」という形にする必要があります。下の手順はそのための回避策です（端末で1回だけ）。

## 導入（端末で1回だけ）

### 1. コードをコピー
[`bookmarklet.txt`](bookmarklet.txt) の中身（`javascript:` で始まる**1行**）を全文コピーします。
- GitHub の Raw 表示で開き、長押し →「すべてを選択」→「コピー」。
- うまく選べない時は、メモアプリ等に一度貼り付けてから全選択コピーすると確実です。

### 2. ダミーのブックマークを作る
1. Safari で**任意のページ**（このページでも可）を開く。
2. 共有ボタン □↑ →「**ブックマークを追加**」→ 保存。

### 3. ブックマークを編集してコードを貼り付け
1. Safari のブックマーク 📖 →「**編集**」。
2. 手順2で作ったブックマークをタップ。
3. **名前**を「**LapClip Visualizer**」などに変更。
4. **URL 欄の中身を全部消し**、手順1でコピーした `javascript:…` を**貼り付け**。
5. 「完了」。

> 💡 よく使うなら「**お気に入り**」フォルダに入れておくと、アドレスバーのタップですぐ呼び出せます。

## 使い方
1. Safari で LapClip のリザルトページ（`https://matrix-sports.jp/lap/result.php?...`）を開く。
2. ブックマーク（またはお気に入り）から **LapClip Visualizer** をタップ。
3. ページ上に F1 風タイミング表示が重なります。**個人/チーム切替・更新・「元の表示に戻す」**が使えます。
4. 最新の結果に更新したい時は、公式ページを再読み込みしてから、もう一度ブックマークをタップ（または表示内の「更新」）。

## ショートカット版との違い
| | ブックマークレット（本書） | ショートカット（[`../ios-shortcut/`](../ios-shortcut/README.md)） |
|---|---|---|
| 起動 | お気に入り/ブックマークを1タップ | 共有シートから選択 |
| 導入 | ブックマーク編集に貼り付け | ショートアプリで手動作成 |
| コード | PC 版と完全同一 | 同一 + `completion()` 呼び出し |

どちらも 1 回実行型で、常駐の自動更新はしません（再実行で更新）。

## 制限（β）
- iOS / Safari のバージョン差で挙動が変わることがあります。
- 一部の Safari 設定（コンテンツブロッカー等）で `javascript:` ブックマークが無効化される場合があります。
- スマホ向けの **Card / Focus** 専用表示は今後対応（現状は PC と同じテーブル表示）。

## 更新（メンテナ向け）
オーバーレイのロジックは Web 側 `src/bookmarklet/` が実体です。更新したら：

```bash
npm run build:ios-bookmarklet   # bookmarklet をビルドし ios-bookmarklet/bookmarklet.txt を再生成
```
