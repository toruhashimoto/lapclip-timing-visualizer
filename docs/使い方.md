# LapClip タイミングビジュアライザー｜かんたん使い方（PC）

LapClip 公式ページを開くと、**F1予選風のタイミング表示が重なって出る**非公式の観戦補助ツールです（まずはPC向け）。

> ※非公式。LapClip・マトリックス社・Tour of Japan・大会運営とは**無関係**です。公式結果は必ず LapClip で確認してください。
> データはあなたのブラウザ内で読み取るだけ。**外部送信・保存・再配信はしません。**

---

## 1. 入れる（PCで最初の1回だけ）
1. ブラウザ（Chrome / Edge / Firefox）に拡張機能 **Tampermonkey** を入れる。
2. 配布された **`lapclip-visualizer.user.js`** を開く → Tampermonkey が出す確認画面で「**インストール**」。

## 2. 使う
- LapClip のリザルトページ（`matrix-sports.jp/lap/result.php?...`）を**開くだけ**。F1風表示が自動で重なります。
- 右上「**元のLapClip表示に戻す**」で、いつでも公式表示に戻せます（公式ページは消えません）。
- **レース種別はページ内容から自動判定**します（個人TT／マススタート／チームTT）。上部の **個人 / チーム** で手動切替もできます。
- ⭐で注目の選手・チームを登録（自分の端末内だけに保存）。

## 2-2. マススタート（ロード／クリテリウム）の見かた
- 着順は**公式リザルトの「位」**に準拠（集団は同タイムのため）。トップ差・周回（`4/6周`等）・スプリント通過（`SP1`等）・周回遅れ（`-N周`）を表示します。
- 右側（PCは横、スマホは上）の「**レース状況**」に、**逃げ／メイン集団／遅れ**の人数とトップ差、残り周回、通過したスプリント地点を表示。⭐登録した選手が**今どの集団にいるか**もひと目で分かります。

## 3. 更新について
- 公式ページを**再読み込み**すると最新の内容を読み直します。
- 設定で「自動リロード」をONにすると一定間隔（最短30秒）で自動更新も可能。

## 4. チームTT（大鹿）
- チーム別に Lap1/2/3・トップ差・順位を表示します。
- 公式ページに周回タイムが無いときは「**FINISH only**」と出ます（推測の数字は出しません）。

## 5. こまったら
- 表示が出ない／古い → 公式ページを再読み込み。
- それでもダメなら、Tampermonkey が有効か・対象ページ（result.php）かを確認。

> スマホ版（Android アプリ / iPhone ショートカット）は準備中です。まずはPCでどうぞ。🚴
